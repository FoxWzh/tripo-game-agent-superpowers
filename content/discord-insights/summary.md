# Discord Insights

## 一句话洞察

Tripo 已经有最好的 3D 生成原子能力，但用户买的不是能力，是产物。当前 Studio 是工具集，用户脑子里装的是完整工作流。

## 三层证据

### A. 技术词翻译回目标词

- PBR 缺失 -> Unity/UE 游戏资产
- 按颜色分件导出 -> 多色 3D 打印
- ARKit 52 blendshape -> VTuber/AR 虚拟形象
- Quad Mode 回来 -> low-poly game-ready 资产

### B. How-to 和卡住帖

- How do I generate the perfect Rig?
- How to create HD Model?
- How Can I Create Modular Outfit Parts Based on an Existing Character Model?
- I generated a model and can't see/download/import it.

共同点：用户能描述目标，但描述不出步骤。

### C. 用户群体稳定聚类

- 游戏开发者：Unity/UE/Roblox，quad mesh + rig + FBX
- 3D 打印玩家：分件、支撑、颜色分离
- VTuber/虚拟形象：VRM + blendshape
- 内容创作者/独立美术：场景、modular asset、批量产出

稳定群体意味着复合 Skill 可以模板化。
