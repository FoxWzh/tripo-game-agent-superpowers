# Roadmap

## MVP / 4-6 周

- 3 个高价值复合 Skill：`GameReadyCharacter`、`PrintableModel`、`VRMAvatar`
- 意图路由：Tag-based confidence prior
- 成本透明：执行前 credit、时间、风险步骤
- 分级执行：Draft / No-rig / Full
- 基础 Eval：任务级 golden set 30 条

## Phase 2

- 可对话改稿：局部 mesh、局部 texture、局部 rig 修复
- Asset Memory：风格、拓扑预算、命名规范、导出偏好
- 失败自愈：重试、降级、换路径
- Eval 升级：Skill-level 和 end-to-end

## Phase 3

- DCC Bridge：SendToBlender / SendToUnity 自动化
- Modular 资产包工作流
- 场景级生成
- Agent API：把复合 Skill 暴露给开发者生态
